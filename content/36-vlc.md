# VLC Media Player

## What is it?

The best media player for Linux (and any OS). Plays literally every video and audio format — MP4, MKV, AVI, MP3, FLAC, DVDs, streams. No need to install separate codec packs like on Windows.

---

## Installation

```bash
sudo apt install vlc
```

---

## Usage

```bash
# Launch VLC
vlc

# Play a file from terminal
vlc /path/to/video.mp4

# Play a YouTube video (paste URL)
vlc https://youtube.com/watch?v=...
```

> **💡 Tip:** VLC can also stream network content, play DVDs, and convert between media formats — all from one app.
