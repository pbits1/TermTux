# Flameshot (Screenshot Tool — Snipping Tool Alternative)

## What is it?

Best screenshot tool for Linux. Lets you annotate, draw arrows, blur areas, and copy to clipboard — just like Snipping Tool in Windows but much better.

---

## Installation

```bash
sudo apt install flameshot
```

---

## Usage

```bash
# Take a screenshot (opens interactive capture)
flameshot gui
```

> **💡 Tip:** Bind `flameshot gui` to the **Print Screen** key in **Settings → Keyboard → Custom Shortcuts** for quick access!
