# JAVA (JDK)


  What is it?
    Programming language used for Android apps, enterprise software,
    and Minecraft. You need the JDK (Java Development Kit) to compile
    Java code, or just the JRE to run Java apps.

    # Install OpenJDK (free, open-source Java):
    sudo apt install default-jdk

    # Check version:
    java --version
    javac --version

    # Install a specific version:
    sudo apt install openjdk-21-jdk

    # Switch between Java versions:
    sudo update-alternatives --config java
