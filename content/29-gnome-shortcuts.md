# ESSENTIAL KEYBOARD SHORTCUTS (GNOME Desktop)


    Super (Win key)           → Open Activities overview
    Super+L                   → Lock screen
    Super+D                   → Show desktop
    Super+A                   → Open application menu
    Super+Left/Right          → Snap window to left/right half
    Super+Up                  → Maximize window
    Super+Down                → Restore/minimize window
    Alt+Tab                   → Switch between open windows
    Alt+F4                    → Close current window
    Ctrl+Alt+T                → Open terminal
    Ctrl+Alt+Delete           → Power off/restart menu
    Print Screen              → Screenshot (full screen)
    Shift+Print Screen        → Screenshot (selected area)
    Super+Shift+Print Screen  → Screenshot (specific window)
