# OBS Studio (Screen Recording & Streaming)

## What is it?

Open Broadcaster Software. The #1 tool for screen recording and live streaming (Twitch, YouTube). Free, open-source, and professional quality.

---

## METHOD 1 — Install via PPA (Latest Version)

```bash
sudo add-apt-repository ppa:obsproject/obs-studio
sudo apt update
sudo apt install obs-studio
```

---

## METHOD 2 — Via Flatpak

```bash
flatpak install flathub com.obsproject.Studio
```

---

## Launch

```bash
obs
```
