# Snap Package Management

Snap packages are self-contained apps that include all their dependencies. They update automatically and are sandboxed for security.

---

## `sudo snap refresh`

Update all installed snap packages to their latest versions.

```bash
sudo snap refresh
```

---

## `snap list`

List all installed snap packages with versions.

```bash
snap list
```
