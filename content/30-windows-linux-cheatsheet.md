# Windows → Linux Translation Cheat Sheet

| WINDOWS | UBUNTU/LINUX |
| --- | --- |
| File Explorer | Nautilus (Files app) |
| Task Manager | System Monitor / htop |
| Control Panel | Settings app |
| CMD / PowerShell | Terminal (Bash) |
| Event Viewer | journalctl |
| Task Scheduler | cron / systemd timers |
| .exe installer | .deb package / apt install |
| Registry Editor | gsettings / dconf-editor |
| C:\Users\You | /home/username |
| C:\Program Files | /usr/bin or /opt |
| Recycle Bin | ~/.local/share/Trash |
| Device Manager | lspci / lsusb / lshw |
| Disk Management | gnome-disks / fdisk / lsblk |
| Windows Update | sudo apt update && sudo apt upgrade |
| Microsoft Store | Ubuntu Software / Snap Store |
| .zip | .tar.gz / .zip |
| Notepad | gedit / nano |
| Paint | Drawing / GIMP |
| ipconfig | ip a |
| netstat | ss -tuln |
| dir | ls |
| cls | clear |
| copy / xcopy | cp / cp -r |
| move / ren | mv |
| del | rm |
| mkdir | mkdir |
| type | cat |
| findstr | grep |
| tasklist | ps aux |
| taskkill | kill / killall |
