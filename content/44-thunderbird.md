# Thunderbird (Email Client — Outlook Alternative)

## What is it?

Free email client by Mozilla (makers of Firefox). Supports Gmail, Outlook, Yahoo, and any IMAP/POP3 email. Calendar included.

---

## Installation

```bash
# Usually pre-installed. If not:
sudo apt install thunderbird
```

---

## Launch

```bash
thunderbird
```
