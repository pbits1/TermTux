# Useful Tools to Install

Must-have utilities that make Ubuntu even better. Install them all in one go!

---

## htop — Better Task Manager

Interactive, colorful process viewer.

```bash
sudo apt install htop
```

---

## neofetch — System Info Display

Show system info in a pretty format (great for screenshots!).

```bash
sudo apt install neofetch
```

---

## tldr — Simplified Help

Simplified command help with real-world examples.

```bash
sudo apt install tldr
```

---

## ncdu — Disk Usage Analyzer

Interactive disk usage analyzer.

```bash
sudo apt install ncdu
```

---

## tree — Folder Structure Viewer

Show folder structure as a tree.

```bash
sudo apt install tree
```

---

## net-tools — Classic Networking

Classic networking tools (ifconfig, netstat).

```bash
sudo apt install net-tools
```

---

## gnome-tweaks — Desktop Customization

Advanced GNOME desktop customization.

```bash
sudo apt install gnome-tweaks
```

---

## dconf-editor — Hidden Settings Editor

GUI to edit hidden GNOME settings (like Windows Registry Editor).

```bash
sudo apt install dconf-editor
```

---

## timeshift — System Restore

System restore / backup tool (like System Restore in Windows).

```bash
sudo apt install timeshift
```

> **⚠️ HIGHLY RECOMMENDED** — Set up weekly snapshots!

---

## gparted — Partition Manager

GUI partition manager (like Disk Management in Windows).

```bash
sudo apt install gparted
```

---

## vlc — Media Player

Best media player — plays everything.

```bash
sudo apt install vlc
```

---

## gimp — Image Editor

Image editor (Photoshop alternative).

```bash
sudo apt install gimp
```

---

## stacer — System Optimizer

System optimizer and monitor (like CCleaner for Linux).

```bash
sudo apt install stacer
```
