# GIMP (Image Editor — Photoshop Alternative)

## What is it?

GNU Image Manipulation Program. Free and open-source image editor. Supports layers, masks, filters, brushes — almost everything Photoshop can do. Great for photo editing, graphic design, and memes.

---

## Installation

```bash
sudo apt install gimp
```

---

## Usage

```bash
# Launch GIMP
gimp

# Open an image directly
gimp /path/to/image.png
```
