# Telegram

## What is it?

Fast, secure messaging app. The Linux desktop client works great and syncs with your phone.

---

## METHOD 1 — Via Snap (Easiest)

```bash
sudo snap install telegram-desktop
```

---

## METHOD 2 — Via Flatpak

```bash
flatpak install flathub org.telegram.desktop
```

---

## Launch

```bash
telegram-desktop
```
