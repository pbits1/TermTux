# Steam (Gaming)

## What is it?

Valve's game store and launcher. Many Windows games now work on Linux through Proton (Steam's compatibility layer). AAA games like GTA V, Elden Ring, etc. run on Linux now.

---

## Installation

```bash
sudo apt install steam
```

> **💡 Tip:** First launch will download updates (~350MB). Enable "Steam Play" in Settings → Compatibility to run Windows games.

---

## Launch

```bash
steam
```

---

## Enable Proton for All Games

To play Windows games on Linux:

**Steam → Settings → Compatibility → "Enable Steam Play for all titles"**

> **💡 Tip:** Check [ProtonDB](https://www.protondb.com) to see which games work well with Proton on Linux.
